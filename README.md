# Echo Engineers — Community Website

> Building Future Innovators Through Technology
> Student-led tech community at SOET, MUIT Noida

## Quick Start

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # Production build
```

## Tech Stack
- Vite + React
- Tailwind CSS v3
- Framer Motion
- React Icons
- Google Fonts: Syne + DM Sans

## Structure
src/components/: Navbar, Hero, About, WhatWeDo, Events, Projects, Team, Stats, Gallery, Testimonials, JoinCommunity, Contact, Footer, Utilities
